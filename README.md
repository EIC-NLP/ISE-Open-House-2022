# ISE-Open-House-2022\_\_Snake-CV-Game

## How to Run

1. Clone the repository
2. cd to the project folder then `pip install -r requirements.txt`

**main.ipynb** for development using jupyter notebook. No need to run the whole notebook, when debugging.

**main.py** for production. Run the whole notebook to generate the final output

3. Change the `CAMERA_PORT` to whatever the port your webcam is connected to. best guess is 0 to 3.
